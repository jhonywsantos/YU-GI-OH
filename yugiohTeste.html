GAME GUIDE

🎮 GAME LOGIC - YU-GI-OH DUEL

Attack Comparison:

  • If Player's ATK > Bot's ATK → Player wins the round
  • If Bot's ATK > Player's ATK → Bot wins the round
  • If ATK values are equal → Round ends in a draw

DAMAGE CALCULATION (Based on Defense)

After identifying the winner by ATK, compare the loser's DEF with the winner's ATK:

  • If loser's DEF ≥ winner's ATK → Loser loses 10% life
  • If loser's DEF < winner's ATK → Loser loses 20% life
  • ATK draws don't result in life loss for either player

GENERAL RULES

  • Life is only reduced if there's a round winner
  • Each round (except draws) results in life loss for someone

END GAME CONDITIONS (after 5 rounds)

  • Player with most remaining life points wins
  • If life is tied, winner is decided by most round wins
  • If still tied, result is considered a total draw
  • Defeated player's life bar visually ends at 0% (even if technically higher) for visual impact
